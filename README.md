# Institutional Census Dashboard

This Power BI dashboard visualizes student enrollment, degrees awarded, and retention rates using sample census data. It includes interactive charts, slicers, and DAX measures.

## Key Features

- Total Students, Total Enrollments, Degrees Awarded (Cards)
- Enrollment by Major and Term (Bar Charts)
- Degree Distribution (Pie Chart)
- Retention Rate % (Gauge)
- Interactive Slicers for filtering by Term, Major, and Degree
